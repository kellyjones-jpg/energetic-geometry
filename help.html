2. PV Technology — Light Distortion Patterns

Visual PV Technology Warp Style Visual Strategy Conceptual Rationale
Monofacial PV Linear Warp Shapes bend one way; converging lines; stable downward tilt Mimics single-direction light collection with lens-like planar flow
Bifacial PV Symmetric Warp / Mirror Bilateral waveforms; bulging vertical shapes; moiré overlap effects Reflects dual light intake; symmetry suggests front and rear absorption
Translucent PV Radial / Organic Warp Rippled, circular distortions; transparent mesh with center deformation Suggests permeability, diffusion, greenhouse light conditions

9. Crop Type — Edge Style / Curvature

Visual Crop Type Group Edge Style / Curvature Visual Metaphor / Rationale

Mixed / assorted crops (kale, tomatoes, herbs, beans, squash, etc.) Layered overlays / composite edges Combines multiple shapes and curvature patterns
